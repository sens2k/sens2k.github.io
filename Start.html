Работа с Unity

В этом части вам понадобятся скрипты, ознакомиться с ними вы можете в разделе Scripting

Запустим Unity и создадим проект

Выбираем 2D проект
Даем название
Указываем папку для сохранения

Добавляем нужные папки

В assets создаем 5 папок

Animation - для анимаций
Prefabs - для префабов
Scripts - для C# скриптов
Sprites - для картинок
Textures - для текстур

Создаем игрока

Из папки Sprite переносим на сцену за ранее созданный рисунок

Даем название Player
Добавляем компоненты:

RigidBody2D (физика)
Polygon Collider 2D (границы объекта)
Fly Player (скрип для игрока Scripting)

Создаем врага

В папке texture создаем Sprite Square и переносим его на сцену

Даем название FlyEnemy
Добавляем компоненты RigidBody2D (физика)
Box Collider 2D (границы объекта)
Fly Enemy (скрип для игрока Scripting)

После чего перносм объект со сцены в папку Prefabs, чтобы создавать его копии в дальнейшем

Добавляем вспомогательный элементы

В сцене создаем два объкта: Spawner и GameManager

К Spawner добавляем скрипт FlySpawner (Scripting), после чего в поле meteor переносим наш префаб Enemy

К GameManager добавляем скрипт ScoreaManager (Scripting)

Добавляем Canvas

На сцене создаем элемент Ui - Text

Добавляем скрипт UiManager к элементу Canvas (Scripting), после чего в поле text переносим наш элемент текста.

В инспекторе в поле Render Mode выставляем Screen - Space - Camera, и переносим туда нашу камеру.

Настраиваем text

В инспекторе настраиваем цвет, шрифт, размер

Выравниваем его по экрану
